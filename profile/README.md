# weftspun

**A recommendation engine for user-made 3D worlds** — it suggests the avatars, maps, and props a
creator is likely to want next.

It reads an item's actual **content** (3D shape, textures, images, text), so a brand-new asset can be
recommended the moment it exists — no waiting for click history.

Small, independent **Elixir** programs, each a single self-contained binary. Open source · built for V-Sekai.
